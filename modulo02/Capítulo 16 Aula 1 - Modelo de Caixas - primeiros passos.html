Capítulo 16 Aula 1 – Modelo de Caixas: primeiros passos

Nessa aula, vamos aprender sobre o Modelo de Caixas / Box model. É uma teoria super importante, para entendermos o box-size e a forma como os box são exibidos.

Quando utilizamos um elemento dentro do outro, o nome que damos para essa aplicação é aninhamento!

Uma caixa, ela tem um tamanho, com medidas muito especificas e com palavras que as vezes se parecem muito, ALTURA = HEIGHT e LARGURA = WIDTH

Também, podemos traçar uma linha em volta do nosso conteúdo (da nossa box), essa linha nós chamados de border (borda).

O espaço entre, a borda e o conteúdo, é chamado de padding (que pode ser chamado de acochoamento ou preenchimento), é como se a gente engordasse o elemento, para que a borda fique distante do conteúdo. Padding é o espaço interno de um elementro.

Para um espaço entre elementros, nós temos a margin (margem). Ela nós ajuda e auxilia a distanciar elementos diferentes. Margin é o espaço externo de um elemento.

Temos também o outline (traçado), ele é uma borda, só que fora do elemento.

Tipos de Caixa

O H1, por exemplo é um Tipo de Caixa, temos basicamente, dois tipos de caixas:

  • BOX-LEVEL
    • Um box-level, ele sempre inicia em uma linha nova.
    • O box-level, sempre ocupa a largura total da tela.
    • Sempre que utilizamos uma DIV, estamos utilizando um box, do tipo box-level.
        • Exemplos de BOX-LEVEL
      • <h1 - h6>
      • <p>
      • <main>
      • <header>
      • <nav>
      • <article>
      • <aside>
      • <footer>
      • <form>
      • <video>
  • INLINE-LEVEL
    • O inline-level, não vai pular para uma próxima linha.
    • E depois que finalizado, ele não quebra a linha.
    • Sempre que utilizamos a TAG span, estamos utilizando um box, do tipo inline-level
        • Exemplos de INLINE-LEVEL
      • <span>
      • <a>
      • <code>
      • <small>
      • <strong>
      • <em>
      • <sup> - <sub>
      • <label>
      • <button>
      • <input>
      • <select>
