Capítulo 03 Aula 01 – Como funciona HTML e CSS

Nós, DEVs, desenvolvemos em linguagem HTML e CSS.

A sigla HTML, significa: HyperText Markup Language, que é uma linguagem de MARCAÇÃO de HIPERTEXTOS.

A sigla CSS, significa: Cascading Style Sheets, que são folhas de estilo em Cascata.

O que é HTML?

HTML é tudo o que é Conteúdo em um site.
Ele funciona com a abertura e fechamento das Tags. < >
Exemplo: Textos, Imagens, Vídeos, Tabelas.

O que é CSS?

CSS é o Design da página.

Para alterar o estilo em CSS, utilizamos os seletores.
Dentro desses seletores, nós vamos declarar as caracteristicas que vamos
querer para aquele seletor em especifico. Resumindo: Seletores, tem declarações, e cada declaração tem um par de Propriedade e Valor.

Exemplo: Cores, Sombras, Tamanhos, Posicionamentos.

O que é JS

JavaScript é a linguagem de programação responsável pelas interações do Site!
Exemplo: Menus, Camicases, Popups, Alterações de Cores (Dark/Light).

Estrutura Básica de documento HTML

!DOCTYPE html, com abertura e fechamento de TAG. É o que diz para nosso navegador, que é um documento do tipo HTML.

Dentro da tag HTML, ficara TODO o conteúdo do sinal em HTML.

Todo HTML tem duas áreas bem definidas, que é a HEAD e o BODY.
