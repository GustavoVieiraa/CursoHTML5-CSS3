Representação de Dados

Os dados, desde os mais complexos até os mais simples são representados através de 0 e 1, e são mais conhecidos como Binary digit (ou Bit, o bit é a menor unidade de informação que pode ser armazenada ou transmitida).

E temos também o Byte, o byte é uma sequência de 8 bits. Ele é a porção miníma que pode representar uma determinada informação. Ex. 01000001 = A

Para essa representação de Dados, nós temos os multiplos dos Bytes: KB, MB, GB, TB, etc. Por exemplo: 1024 Bytes = 1KB Lembrando que a referência é sempre 1024! Pois nós, seres humanos utilizamos a base 10 normalmente em nosso dia a dia para realizar nossas conversões e achar os multiplos seja para Kilograma, Metros, etc. Mas, o computador não, ele utiliza, a base 2. Por isso é sempre 1024! Por exemplo:

    1024 Bytes = 1KB 1024 KB = 1 MB > 1024 MB = 1 GB > 1024 GB = 1 TB > 1024 TB = 1 PB > 1024 PB = 1 EB > 1024 EB = 1 ZB > 1024 ZB = 1 YB >

Observação: MB e Mb são DIFERENTE!
MB com o B maiusculo: é sobre ARMAZENAMENTO!
Mb com o b minusculo: é sobre VELOCIDADE DE TRANSFERENCIA!
Exemplos: Comprei um PenDrive com 8MB, é armazenamento! Minha internet em casa, a velocidade é de 400Mb é sobre velocidade de Transferencia!
